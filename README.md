# 📅 Planner Semanal Pro + Cronograma Inteligente

O **Planner Semanal Pro** é um organizador de tarefas avançado desenvolvido com tecnologias web nativas. Esta versão foca na gestão precisa do tempo, permitindo o agendamento de atividades com horários específicos e ordenação cronológica automática.

---

## 🌟 O que há de novo? (Versão 2.0)

Diferente de uma lista de tarefas comum, este planner agora conta com:
* **Gestão de Horários:** Campo dedicado para definir o horário de cada atividade.
* **Ordenação Cronológica:** O sistema organiza automaticamente suas tarefas pelo horário, garantindo que você veja o que precisa ser feito primeiro.
* **Badges de Tempo:** Identificação visual rápida do horário e do dia da semana.
* **Confirmações de Segurança:** Prevenção contra exclusões acidentais.

---

## 🚀 Funcionalidades Principais

* **Filtro por Dia da Semana:** Alterne entre os dias para focar apenas no que importa no momento.
* **Barra de Progresso Dinâmica:** Feedback visual da sua produtividade semanal em tempo real.
* **Prioridades Coloridas:** Categorização visual em Alta, Média e Baixa prioridade.
* **Persistência (LocalStorage):** Seus dados ficam salvos no navegador; nada se perde ao atualizar a página.
* **Totalmente Responsivo:** Utilize no computador, tablet ou smartphone sem perda de usabilidade.

---

## 🛠️ Tecnologias e Conceitos Aplicados

* **HTML5:** Estrutura semântica e inputs nativos de tempo e seleção.
* **CSS3 (Modern UI):** * **CSS Grid & Flexbox:** Para um layout adaptável e organizado.
    * **Variáveis CSS:** Para manutenção simplificada de cores e temas.
    * **Animações (@keyframes):** Para transições suaves na interface.
* **JavaScript (ES6+):**
    * **Array.sort():** Algoritmo para ordenar tarefas por hora.
    * **Array.filter():** Gerenciamento dinâmico da exibição por dia.
    * **JSON Serialization:** Persistência de objetos complexos no `localStorage`.

---

## 📂 Como Executar o Projeto

1.  Baixe o arquivo `index.html`.
2.  Abra-o em qualquer navegador (Chrome, Firefox, Safari ou Edge).
3.  **Dica:** Adicione o arquivo aos seus favoritos para usá-lo como seu planner oficial no dia a dia.
